# Nuxt 3 Boilerplate example

- Nuxt3
- Pinia
- Scss
- Eslint
- Prettier
- TailwinCSS + PostCSS

# Installation

````
$ pnpm install --shamefully-hoist
```

# Run server

## Dev

```bash
$ pnpm dev
```

## Production

```bash
$ pnpm build
```

Locally preview production build:

```bash
$ pnpm run preview
```
````
